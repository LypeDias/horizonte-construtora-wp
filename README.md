# Horizonte Construtora — Plataforma Web

Projeto desenvolvido utilizando WordPress com foco na apresentação institucional de uma construtora, incluindo páginas informativas, sistema de busca de empreendimentos e integração com ferramentas de contato.

## Funcionalidades

* Header com menu interativo
* Footer estruturado
* Página Home com sistema de pesquisa de obras (em desenvolvimento)
* Página Empreendimentos com sistema de pesquisa (em desenvolvimento)
* Página Sobre com formulário de contato e integração com mapa
* Página Contato com formulário funcional e integração com mapa
* Botão de contato via WhatsApp integrado ao site

## Plugins Utilizados

* Elementor
* MetForm
* Search & Filter
* Ultimate Addons for Elementor (UAE)
* Custom Post Type UI
* Click to Chat

## Tecnologias Utilizadas

* WordPress
* Elementor (Page Builder)
* PHP
* HTML, CSS e JavaScript

## Estrutura do Projeto

Este repositório contém a exportação do conteúdo do WordPress:

export.xml

## Como Executar o Projeto

1. Instalar o WordPress em um ambiente local ou servidor
2. Instalar o tema Hello Elementor
3. Instalar os plugins listados acima
4. Acessar o painel do WordPress
5. Ir em Ferramentas → Importar
6. Importar o arquivo export.xml

## Observações

* Algumas funcionalidades ainda estão em desenvolvimento, como o sistema completo de pesquisa de empreendimentos
* O layout foi desenvolvido utilizando o Elementor, portanto depende dos plugins instalados para funcionamento completo
